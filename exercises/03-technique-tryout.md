# Übung 3 (Kapitel 7) — Eine Technik ausprobieren

**Zeit:** 10 min (nach Lightning Round, Block 6)  
**Modus:** Paare  
**Ziel:** **Eine** Arbeitsweise aus der Lightning Round kurz anwenden

## Aufgabe

1. Wählt **eine** Technik aus Block 6 (z. B. Review-first, Plan-first, TDD).
2. Schreibt einen **kurzen Prompt** dafür — Code: `sample/orderCalculator.js` oder eigenes Szenario.
3. Optional: einmal in Copilot ausführen und mit `node --test orderCalculator.test.js` verifizieren.
4. Notiert in einem Satz: „Hat die Technik geholfen — warum (ja/nein)?“

## Mini-Gerüst

```text
Technik: [Name aus Lightning Round]
Prompt:
[2–6 Zeilen, inkl. klarer Action + Constraint]
Ergebnis (1 Satz):
```

## Debrief (Trainer, optional 1–2 min)

- Welche Technik war am überraschendsten nützlich?
- Merke: Katalog ≠ Pflicht — Montag die Technik wählen, die zur Aufgabe passt.
